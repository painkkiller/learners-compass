# Learner’s Compass – дашборд ученика

**Краткое описание**  
Сервис, который показывает ученику его прогресс по курсам, ближайшие дедлайны и ведёт к следующему уроку. Первая итерация — минимальный дашборд без авторизации, но с данными-заглушками.

## Учебный срез этого модуля
- **Узкий фокус первой итерации**:  
  «Просмотр курсов, прогресса по каждому, список дедлайнов и кнопка перехода к следующему уроку».
- **Почему этот срез подходит для ограниченной сессии**:  
  Не требует авторизации, сложной базы данных или внешних API. Все данные — моковые. Результат — работающий веб-интерфейс за 1-2 часа с агентом.
- **Артефакты для агента**:  
  `AGENTS.md` (постоянные правила), `docs/iteration-1-plan.md` (пошаговый план), `docs/context.md` (минимальный контекст).

## Как запустить

```bash
git clone <repo-url>
cd learners-compass/frontend
npm install
npm run dev
```

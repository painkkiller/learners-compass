# Минимальный контекст для агента (Learner’s Compass)

## Что уже есть в проекте (стартовое состояние)
- Пустой репозиторий с этой структурой.
- Никакого кода, кроме этих документов.

## Что агенту нужно прочитать перед началом (в порядке приоритета)
1. `AGENTS.md` — постоянные правила.
2. `docs/iteration-1-plan.md` — пошаговый план.
3. `README.md` — общая цель.

## Технические предположения (чтобы не гадать)
- **Фронтенд**: React + Vite. Точка входа: `frontend/src/main.jsx`.
- **Бэкенд**: Express. Точка входа: `backend/server.js`. Порт 3001.
- **Тесты**: Jest + React Testing Library. Команда: `npm test` в папке `frontend/` и `backend/`.
- **Стили**: CSS-модули (или просто CSS — агент выбирает минимальное).

## Что НЕ нужно трогать
- `scripts/` — пусто, не создавать там файлы без запроса.
- `infra/` — нет такой папки, не создавать.
- База данных — не подключать, использовать моковые данные.

## Как проверить прогресс без агента
- Открыть `http://localhost:5173` (Vite) — виден дашборд.
- Открыть `http://localhost:3001/api/courses` — виден JSON.
- Запустить `npm test` — зелёный.
